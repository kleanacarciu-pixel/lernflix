# Infos & Material für die Website (lernemitanna.de)

Lege hier alles ab, was für die Website wichtig ist – ich lese es aus und baue es ein.

## Was hier reingehört
- **Design / Vorlage:** deine PDF, Screenshots, Skizzen, Moodboard
- **Logo:** als PNG/SVG (gern mit transparentem Hintergrund)
- **Fotos:** Hero-Foto, Bilder von dir, vom Unterricht usw.
- **Texte:** Startseite, Über mich, Leistungen, Preise, FAQ …
- **Marke:** gewünschte Farben (z. B. #2BB3C0) und Schriften
- **Kontakt:** E-Mail, Telefon, Social-Media-Links, Adresse

## So lädst du Dateien hoch (über die GitHub-Website)
1. Diesen Ordner `lernemitanna-theme/inhalte/` auf GitHub öffnen.
2. Oben rechts **„Add file" → „Upload files"** klicken.
3. Dateien reinziehen → unten **„Commit changes"**.

Danach sag mir kurz Bescheid – ich schaue mir alles an.

> Tipp: Noch einfacher ist es, die Dateien direkt im Chat anzuhängen.
